# Scrappy

#### Requirements
1. NodeJs (12.14.1) - Runtime
2. Cheerio - Parses the data the url provided.


#### Development
Project was setup using `node` version `12.14.1`
``yarn install``
``yarn start``


#### API

Endpoint
`` POST /api/parse ``

Request Body
``
{
  "url": "https://www.amazon.in/dp/B08695ZSP6/"
}
``
